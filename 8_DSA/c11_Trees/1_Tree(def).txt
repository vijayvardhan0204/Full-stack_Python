What is a Tree (Data Structure)?

A tree is a hierarchical data structure made of nodes used to represent hierarchical relationships.

Root → top node

Parent → node with children

Child → node under a parent

Leaf → node with no children


Types of Trees (Important)

Binary Tree – max 2 children

Binary Search Tree (BST) – left < root < right

Heap – complete tree (min / max)

AVL Tree – self-balancing BST


Why Trees Are Used?

File systems

Databases

Searching & sorting

AI & decision making
